# 生產力方法全攻略

---

## 一、專注力（Focus）

### 1. 時間區塊法（Time Blocking）
- 將一天切成 **專注區塊**（90 分鐘）+ **休息區塊**（15-20 分鐘）
- 每個區塊只做一件事，不切換任務
- 上午黃金時段（9-12 點）留給最難的工作

### 2. 番茄鐘（Pomodoro Technique）
- 25 分鐘專注 → 5 分鐘休息 → 每 4 輪休息 15-30 分鐘
- 適合不容易進入狀態的人，降低啟動門檻
- 計時器響之前**絕不中斷**

### 3. 深度工作（Deep Work）
- 關閉所有通知、手機翻面、關閉無關分頁
- 設定「勿擾時段」，告知同事這段時間不回訊息
- 環境固定：同一張桌子、同一首背景音樂 → 建立條件反射

### 4. 單任務原則（Single-Tasking）
- 多工處理是生產力殺手，每次切換任務損失 **23 分鐘**恢復時間
- 一次只開一個工作視窗
- 用「停車場筆記」記下臨時冒出的想法，之後再處理

### 5. 消除干擾清單
- 手機：開啟專注模式 / 放到另一個房間
- 電腦：用 Cold Turkey / Freedom 封鎖社群媒體
- 環境：耳塞或白噪音、整理桌面只留必要物品
- 人際：設定「可被打擾」的明確時段

## 二、動力與動機（Motivation）

### 1. 兩分鐘法則（2-Minute Rule）
- 任何任務，先只做 **2 分鐘**
- 重點不是完成，是**啟動**——一旦開始，慣性會帶你繼續
- 「我只寫一行 code」→ 往往寫了一小時

### 2. 目標視覺化
- 把目標寫在每天看得到的地方（桌面便條、手機桌布）
- 想像完成後的感覺，而非過程的痛苦
- 用進度條追蹤完成百分比（視覺回饋增強動力）

### 3. 獎勵機制
- 完成困難任務後給自己**即時獎勵**（咖啡、散步、一集影片）
- 大目標拆成小里程碑，每個里程碑都慶祝
- 避免獎勵與目標衝突（如：健身後不要用垃圾食物當獎勵）

### 4. 問責機制（Accountability）
- 找一個夥伴互相報告進度
- 公開承諾（社群、朋友圈）增加違約成本
- 用 Beeminder 類工具：沒達標就扣錢

### 5. 內在動機三要素（Self-Determination Theory）
- **自主感**：自己選擇做什麼、怎麼做
- **勝任感**：任務難度匹配能力（不太難也不太簡單）
- **歸屬感**：和志同道合的人一起努力

## 三、精力管理（Energy Management）

### 1. 睡眠（最重要的生產力工具）
- 固定時間上床 + 起床，週末也不例外
- 睡前 1 小時無螢幕（藍光抑制褪黑激素）
- 目標 7-8 小時，少於 6 小時認知能力下降 30%+

### 2. 運動
- 每天至少 30 分鐘中等強度運動
- 運動後 2-3 小時是認知能力高峰期
- 久坐每 50 分鐘站起來走動

### 3. 飲食
- 避免高糖食物造成血糖崩盤（sugar crash）
- 咖啡最佳飲用時間：起床後 90 分鐘（皮質醇下降後）
- 午餐七分飽，避免飯後昏沉

### 4. 能量週期
- 追蹤自己一天的精力曲線，找出高峰與低谷
- 高峰期做創造性 / 困難工作
- 低谷期做機械性工作（回信、整理、行政）

## 四、任務管理（Task Management）

### 1. GTD（Getting Things Done）
- **收集**：所有待辦事項丟進收件匣（Inbox）
- **處理**：逐一判斷→ 2 分鐘內能做完就立刻做；否則分類
- **組織**：分到「下一步行動 / 等待中 / 專案 / 未來也許」
- **回顧**：每週檢視所有清單
- **執行**：根據情境、時間、精力選擇下一步

### 2. 艾森豪矩陣（Eisenhower Matrix）

|  | 緊急 | 不緊急 |
|---|---|---|
| **重要** | 立刻做 | 排時間做（最有價值） |
| **不重要** | 委託別人 | 刪除 |

- 大多數人卡在「緊急但不重要」，忽略「重要但不緊急」
- 每天最多 3 個「重要」任務

### 3. MIT（Most Important Tasks）
- 每天開始前選出 **1-3 個最重要的任務**
- 在一天最有精力的時候優先完成
- 完成 MIT = 這一天就是成功的

### 4. 任務拆解（Task Decomposition）
- 大任務令人拖延，拆成 **30 分鐘以內**可完成的子任務
- 每個子任務要有明確的「完成定義」
- 範例：「寫報告」→「列出大綱」→「寫第一段」→「找數據」→「校稿」

### 5. 批次處理（Batching）
- 同類任務集中處理：統一回信、統一開會、統一寫文件
- 減少情境切換成本
- 設定固定時段處理信件（如 10:00、14:00、17:00）

## 五、習慣養成（Habit Building）

### 1. 原子習慣四法則（Atomic Habits）
- **提示明顯**：把跑鞋放在門口、把書放在枕頭上
- **有吸引力**：綁定喜歡的事（聽 podcast 只能在運動時聽）
- **容易執行**：降低門檻（「每天讀 1 頁」比「每天讀 1 小時」容易開始）
- **令人滿足**：完成後打勾、連續紀錄（streak）

### 2. 習慣堆疊（Habit Stacking）
- 「做完 A 之後，我就做 B」
- 範例：「倒完咖啡後，我就打開筆記寫今天的 3 個 MIT」
- 利用既有習慣當作新習慣的觸發器

### 3. 身份認同驅動
- 不要想「我要養成讀書的習慣」
- 改成「我是一個愛學習的人」
- 每次行動都是在為你想成為的人投票

### 4. 不要斷鏈（Don't Break the Chain）
- 每天完成目標就在日曆上打 X
- 連續天數越長，你越不想中斷
- 允許偶爾 miss，但**絕不連續 miss 兩天**

## 六、心理策略（Mental Strategies）

### 1. 帕金森定律（Parkinson's Law）
- 工作會膨脹到填滿所分配的時間
- 解法：**人為設定更緊的截止日期**
- 「這份報告 3 天內交」→ 給自己 1.5 天

### 2. 吃掉那隻青蛙（Eat the Frog）
- 每天第一件事做**最困難、最抗拒**的任務
- 完成後一整天都很輕鬆
- 拖延的東西只會越拖越難

### 3. 對抗完美主義
- 「完成比完美重要」（Done is better than perfect）
- 先產出 80% 的粗糙版本，再迭代優化
- 設定「最低可接受標準」，達到就停手

### 4. 心流狀態（Flow State）
- 條件：目標明確 + 即時回饋 + 難度略高於能力
- 進入心流需要 15-20 分鐘不被打斷
- 追蹤什麼條件讓你最容易進入心流，然後複製

### 5. 正念與冥想
- 每天 10 分鐘冥想可顯著提升專注力
- 工作中感到焦慮時：深呼吸 4-7-8（吸 4 秒、憋 7 秒、吐 8 秒）
- 正念 = 覺察當下，不評判 → 減少反芻思維

## 七、工具與系統（Tools & Systems）

### 1. 筆記系統
- **Zettelkasten（卡片盒筆記法）**：一張卡片一個想法，互相連結
- **PARA**：Projects / Areas / Resources / Archives 四層分類
- 工具選擇：Obsidian、Notion、HackMD（選一個堅持用）

### 2. 日常模板

**晨間計畫（5 分鐘）：**
- [ ] 今天的 3 個 MIT 是什麼？
- [ ] 今天的時間區塊安排？
- [ ] 有什麼可能的干擾？如何預防？

**晚間回顧（5 分鐘）：**
- [ ] 今天完成了什麼？
- [ ] 什麼做得好？什麼可以改善？
- [ ] 明天最重要的一件事？

### 3. 自動化一切重複性工作
- Git hooks、CI/CD、腳本自動化
- 文字展開（Text Expander）處理常用回覆
- 鍵盤快捷鍵：花 1 小時學，省 100 小時

## 八、避免常見陷阱

| 陷阱 | 解法 |
|---|---|
| 過度規劃不行動 | 規劃不超過每天 10 分鐘，剩下的時間執行 |
| 一次改變太多習慣 | 一次只加一個新習慣，穩定 2 週後再加下一個 |
| 忽略休息 | 休息是生產力的一部分，不是偷懶 |
| 比較心態 | 只跟昨天的自己比，不跟別人比 |
| 工具焦慮 | 最好的工具是你會持續用的那個 |
| 忙 ≠ 有產出 | 問自己「這件事完成後，有什麼改變？」 |
| 說 Yes 太多 | 每說一個 Yes，就是對其他事說 No |

## 九、關鍵公式

```
生產力 = 專注力 × 精力 × 時間 − 干擾
```

- **專注力**：單任務、深度工作、消除干擾
- **精力**：睡眠、運動、飲食、能量管理
- **時間**：時間區塊、批次處理、說 No
- **干擾**：手機、通知、多工、完美主義

> 不要試圖做更多事，而是做對的事，並且做好。
